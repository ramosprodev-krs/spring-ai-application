**Project Start Date: July 2026**  

This project was developed in **Java** using the **Spring Boot 4.1.0**.
An intelligent expense management application that leverages **Spring AI** and **Google Gemini** for voice-activated expense tracking, with secure authentication and data persistence.

### Technologies used:    
- Java 21 (Spring Framework)  
- Spring AI 2.0.0-M4
- Spring AI Google Gemini Integration
- Spring Security & JWT
- Spring Data JPA
- PostgreSQL
- React (TypeScript)
- Lombok
- Jackson
- Spring Boot Validation
